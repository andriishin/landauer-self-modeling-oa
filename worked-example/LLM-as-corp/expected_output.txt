Refounded quantity: eta_v = I_pred / I_mem in [0, 1]. The model is
predictively strong (large I_pred) but its payment loop is open;
demarcation is carried by S, not by any energetic-efficiency ratio.

Counterfactual ablation over nested boundaries B1 < B2 < B3 < B4:
  B  components                                    window        S  self-paying subject
  ----------------------------------------------------------------------------------------
  B1 weights only                                  static        0  none (degenerate)
  B2 weights + KV-cache + executing server         session       0  none (grid pays)
  B3 B2 + training pipeline (data + GPU park + RLHF)model gen.    1  pipeline (not model)
  B4 B3 + corporation (legal entity)               generations   1  firm (not model)

Per-boundary reasoning:
  B1 (S = 0): counterfactual-degenerate: static object with the server off
  B2 (S = 0): grid pays holding/updating; no physical error-return channel to weights
  B3 (S = 1): own GPU dissipation appears; error-return loop closes on the operators
  B4 (S = 1): self-funding loop fully closed -- but the subject is the firm

Verdict: the LLM MODEL is never vital. At B1, B2 the self-payment
predicate fails (S = 0): the grid pays for holding/updating the model
and there is no physical channel returning the model's error to its
carrier. At B3, B4 the loop closes (S = 1), but the self-paying
subject is no longer the model -- it is the pipeline, then the firm.
Extending the boundary re-addresses the subject; it does not confer
vitality on the model.

eta_v value: NOT defined here for any boundary. A number would require
an MI estimate of I_pred and I_mem of the respective loop -- future
work; no eta^corp figure is asserted.
