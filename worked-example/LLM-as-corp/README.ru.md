# Разобранный пример: структурный вердикт для LLM-as-corporation

Структурный вердикт для крупной языковой модели на вложенных границах
counterfactual ablation $B_1 \subset B_2 \subset B_3 \subset B_4$ под
переоснованной центральной величиной $\eta_v = I_{\text{pred}}/I_{\text{mem}} \in [0,1]$.
Прилагается к § 3.4 + Supplementary § S3.4 (Андрийшин, *Theory in Biosciences*,
в подготовке).

## Содержательный смысл

LLM — сильнейший чистый случай новой демаркации: система с **сильной
предсказательной моделью и $S = 0$**. Обученная модель предсказательно сильна
(большое $I_{\text{pred}}$), но петля её оплаты разомкнута. Демаркацию несёт
предикат самооплаты $S$, **не** энергетическая эффективность; прежнее число
$\eta^{\text{corp}}$ удалено. Расширение границы до $S = 1$ **не** делает *модель*
витальной — оно переадресует субъект самооплаты.

| Граница | Компоненты | Окно | S | Субъект самооплаты |
|---------|-----------|------|---|--------------------|
| $B_1$ | только веса | статика | 0 | нет (вырожденно) |
| $B_2$ | веса + KV-cache + исполняющий сервер | сессия | 0 | нет (платит электросеть) |
| $B_3$ | $B_2$ + обучающий пайплайн (данные + парк GPU + RLHF) | поколение моделей | 1 | пайплайн (не модель) |
| $B_4$ | $B_3$ + корпорация (юрлицо) | поколения | 1 | фирма (не модель) |

- На $B_1, B_2$: $S = 0$ — электросеть платит за удержание/обновление модели, и нет физического канала возврата ошибки модели её носителю.
- На $B_3, B_4$: петля замыкается ($S = 1$), но субъект самооплаты — уже не модель, а пайплайн, затем фирма.

Таким образом, **модель не витальна ни на одной границе**: где $S = 1$, измеряемым
объектом оказывается корпорация, не модель. Значение
$\eta_v = I_{\text{pred}}/I_{\text{mem}}$ для любой из этих петель здесь **не
утверждается** — число потребовало бы MI-оценки $I_{\text{pred}}$ и
$I_{\text{mem}}$ соответствующей петли (будущая работа). Никакого числа
$\eta^{\text{corp}}$ не приводится.

## Содержимое

| Файл | Назначение |
|------|------------|
| `eta_llm_corp.py` | Структурный вердикт на вложенных границах $B_1$–$B_4$ (предикат $S$, субъект самооплаты) |
| `expected_output.txt` | Эталонный вывод (LF) для верификации |

Использует только стандартную библиотеку Python.

## Запуск

```bash
python eta_llm_corp.py
```

## Связь с литературой

Предикат самооплаты $S$ предлагает термодинамическую ко-характеристику структурной
тенденции, изучаемой в AI-safety как instrumental convergence / self-preservation
(Omohundro 2008; Bostrom 2014) и как *embedded agency* (Demski & Garrabrant 2019:
агент как часть среды, оплачивающий собственное вычисление). Пара $(S, \eta_v)$
предлагается как необходимое термодинамическое условие физически реализованного
embedded world-model с собственной петлёй робастности — не как достаточное условие
для конкретных alignment-свойств.

- **Patterson, D.; et al.** Carbon Emissions and Large Neural Network Training. *arXiv:2104.10350* **2021**. (Масштаб инфраструктуры обучения frontier-моделей.)
- **Demski, A.; Garrabrant, S.** Embedded Agency. *arXiv:1902.09469* **2019**. (Агент как часть среды, оплачивающий собственное вычисление.)

## Статус

Вердикт **структурный**: $S = 0$ на $B_1, B_2$ (веса / сессия), $S = 1$ на
$B_3, B_4$ с переадресацией субъекта на пайплайн / фирму. Никакого числа
$\eta^{\text{corp}}$ не утверждается; квантификация $\eta_v$ для петли
пайплайна/корпорации требует отдельной MI-оценки. Самодостаточно: только
стандартная библиотека.
