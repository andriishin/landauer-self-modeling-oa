# Разобранный пример: структурный вердикт для крупного города + демо I_v

Прилагается к § 3.6 (Андрийшин, *Theory in Biosciences*, в подготовке) под
переоснованной центральной величиной $\eta_v = I_{\text{pred}}/I_{\text{mem}} \in [0,1]$.
Самодостаточен (только стандартная библиотека Python), состоит из двух чётко
разделённых частей: структурный вердикт и демонстрация скрининг-индекса $I_v$.

## Структурный вердикт (утверждение § 3.6)

Город **проходит обе оси**:

- $S = 1$ — само-фондируется из собственного бюджета (условие i) и несёт замкнутую
  петлю возврата ошибки: искажение модели потоков возвращается кризисом самому
  городу (условие ii);
- $\eta_v > 0$ — удерживает предсказательную структуру о собственных потоках.

Поэтому город **витален**. **Грейд** — численное значение
$\eta_v = I_{\text{pred}}/I_{\text{mem}}$ среди витальных систем — здесь **не
утверждается**: он требует MI-оценки $I_{\text{pred}}$ и $I_{\text{mem}}$
административного контура (потоки ресурсов → решения), оставленной будущей работе.
Камертонное утверждение несёт **замкнутость петли самооплаты**, а не малость
эффективности. Прежний энергетический знаменатель удалён.

## Демонстрация скрининг-индекса I_v (формула 2b + 2a)

$$I_v = \sqrt[3]{\tilde{T}_v \cdot \tilde{C}_v \cdot \tilde{S}_v} \in [0,1], \tag{2b}$$

где $\tilde{T}_v, \tilde{C}_v, \tilde{S}_v$ — перцентильно-нормированные (формула
2a) ёмкости внутри когорты. Согласно main § 2.5, $I_v$ — **структурный
скрининговый** композит: он **не** делает бинарный тест «жив/нет» (это работа
$\eta_v$), а лишь скринирует структурную готовность. Демо работает на явно
**синтетическом** наборе архетипических профилей и показывает две вещи:

- (2a) перцентильная нормировка отображает сырой прокси в ранг $[0,1]$ относительно когорты;
- (2b) кубический корень (геометрическое среднее) тянет композит к **любой** слабой ёмкости (профиль с одной проваленной осью получает низкий $I_v$, даже если две другие высоки).

Это **не** эмпирический рейтинг реальных городов. Эмпирическая калибровка $I_v$
конкретных городов по открытым данным (OECD FUA, CDP Cities, WIPO/OECD REGPAT,
GDELT, OpenStreetMap) — предмет отдельной работы и здесь намеренно не утверждается.

## Содержимое

| Файл | Назначение |
|------|------------|
| `eta_city.py` | Структурный вердикт ($S=1$, $\eta_v>0$) + демо скрининг-индекса $I_v$ на синтетике |
| `expected_output.txt` | Эталонный вывод (LF) для верификации |

## Запуск

```bash
python eta_city.py
```

Вывод воспроизводит `expected_output.txt`.

## Статус

Вердикт города **структурный** ($S = 1$, $\eta_v > 0$); никакого числа $\eta_v$ не
утверждается. Демо $I_v$ прогоняет формулу (2b)+(2a) на синтетических профилях,
показывая механику скринингового композита; эмпирический рейтинг реальных городов
по $I_v$ не утверждается. Самодостаточно: только стандартная библиотека.
