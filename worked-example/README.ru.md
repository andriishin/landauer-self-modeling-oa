# Разобранные примеры — витальность как самооплаченное самомоделирование

Воспроизводимые приложения к камертонному анализу статьи (Андрийшин,
*Theory in Biosciences*, в подготовке). Каждая папка — самодостаточный пакет
(Python-скрипт + README + эталонный вывод) с привязкой к конкретной секции.

Центральная величина — **предсказательная доля**

$$\eta_v = \frac{I_{\text{pred}}}{I_{\text{mem}}} = 1 - \nu \in [0, 1],$$

отношение двух *информационных* величин по data-processing inequality
($I_{\text{pred}}$ — сколько внутреннее состояние знает о будущем среды,
$I_{\text{mem}}$ — сколько о прошлом), с ностальгией $\nu = 1 - \eta_v$.
Энергетического знаменателя **нет**: витальность — это конъюнкция

$$V(X) \;:=\; S \;\wedge\; (\eta_v > 0),$$

где $S \in \{0,1\}$ — **предикат самооплаты** (система сама оплачивает удержание
своей модели *и* на ней замкнута физическая петля возврата ошибки). Демаркацию
несёт редкость замкнутой петли самооплаты ($S = 1$), а не малость эффективности.
Численное значение $\eta_v$ вычислено полностью только для *E. coli*; для остальных
витальных случаев грейд оставлен будущей MI-оценке.

## Индекс

| Папка | Секция | Вердикт | Что демонстрирует |
|-------|--------|---------|-------------------|
| [`E-coli/`](./E-coli/) | § 3.5 | $S=1$, $\eta_v \sim O(0.1)$ **вычислено** | **Положительный случай**: $\eta_v = I_{\text{pred}}/I_{\text{mem}}$ вычислен точно (гауссов logdet); центральный пример |
| [`thermostat/`](./thermostat/) | § 4.4 | $S=0$, $\eta_v$ не определён | **FEP-граница**: формальная FEP-применимость без самооплаты; центральный анти-FEP аргумент |
| [`city/`](./city/) | § 3.6 | $S=1$, $\eta_v>0$ (грейд не утверждается) | **Социо-техническая система**: замкнутая петля самооплаты; плюс демо структурного скрининга $I_v$ на синтетике |
| [`biosphere/`](./biosphere/) | § 3.7 | $S=1$, $\eta_v>0$ (грейд не утверждается) | **Планетарная система**: единая Гея (или семейство); лестница ёмкостей носителя — явно *не* значение $\eta_v$ |
| [`LLM-as-corp/`](./LLM-as-corp/) | § 3.4 | $S=0$ на $B_1,B_2$; $S=1$ на $B_3,B_4$ | **Расширение границы**: где петля замыкается, субъект самооплаты — фирма, не модель |

## Что не закрыто отдельным расчётом

Структурно проваливающиеся случаи не требуют worked example:

- **Солнце, нейтронная звезда, чёрная дыра** (§ 3.1): $\eta_v = 0$ через **числитель** — нет каузально-релевантной среды, $I_{\text{mem}} = 0$ и $I_{\text{pred}} = 0$.
- **Ураган и пламя** (§ 3.2): $\eta_v = 0$ через **числитель** — диссипация без удерживаемой модели (ставка без модели).
- **Реакция БЖ / автокатализ** (§ 3.2): $S = 0$ — питается, но не корректируется; петля возврата ошибки (условие ii) не замкнута.
- **Кристалл** (§ 3.3): $\eta_v = 0$ через **числитель** — стационарная структура гомоморфна собственной решётке, не будущему сигналу среды ($I_{\text{mem}} \approx 0$).
- **LLM, узкая граница «веса в момент инференса»** (§ 3.4): $S = 0$ — сильная предсказательная модель при разомкнутой петле оплаты; внутриклассовое значение $\eta_v$ на этой границе не определено.

## Запуск всех расчётов

```bash
for dir in E-coli thermostat city biosphere LLM-as-corp; do
    echo "=== $dir ==="
    (cd "$dir" && python eta_*.py)
done
# В E-coli есть также скрипт анализа чувствительности по параметрам:
(cd E-coli && python sensitivity.py)
```

В каждой папке есть `expected_output.txt` (окончания строк LF) для верификации
воспроизводимости наивным diff-ом.

## Зависимости

Все расчёты используют **только стандартную библиотеку Python** (`math`; без
`numpy`, без `scipy`). `pip install` не требуется. Пример *E. coli* считает
гауссовы взаимные информации точно через чистый stdlib-Cholesky log-определитель,
поэтому результат воспроизводим побитово.

## Структура каждой папки

```
<system>/
├── README.md           — описание (англ., по умолчанию): модель, вердикт, статус
├── README.ru.md        — то же на русском
├── eta_<system>.py     — основной расчёт / структурный вердикт
├── sensitivity.py      — анализ чувствительности по параметрам (только E-coli)
└── expected_output.txt — сохранённый вывод для верификации
```

## Статус

Пакет *E. coli* — **иллюстративный worked example** (минимальная линейная модель),
показывающий, что $\eta_v = I_{\text{pred}}/I_{\text{mem}}$ конечна, измерима и
$O(0.1)$, — не калиброванное значение реального организма. Остальные четыре пакета
печатают **структурные вердикты** $(S, \eta_v)$ и величины, выживающие переоснование
(например, $I_{\text{pred}} \le 3$ бит у термостата, потолки ёмкости носителя у
биосферы); численное $\eta_v$ для этих случаев требует отдельной MI-оценки
$I_{\text{pred}}$ и $I_{\text{mem}}$ (будущая работа).
