# Worked Examples for «Landauer Efficiency of Self-Modeling: An Operational Scale of Vitality»

Воспроизводимые расчёты $\eta_L$ и $I_v$ для пяти камертонных систем статьи. Каждая папка — самодостаточный пакет (Python script + README + expected output) с ссылкой на конкретную секцию paper'а.

## Индекс

| Папка | Секция paper'а | $\eta_L$ (центральная оценка) | Что демонстрирует |
|-------|----------------|-------------------------------|-------------------|
| [`E-coli/`](./E-coli/) | § 3.5 | $3 \cdot 10^{-8}$ (ex.), $3 \cdot 10^{-5}$ (comp.) | **Положительный кейс**: бактерия проходит оба порога — central worked example статьи |
| [`thermostat/`](./thermostat/) | § 4.4 | $5 \cdot 10^{-18}$ trivial / $0$ managing | **FEP-граница**: формальная FEP-применимость без self-payment; центральный анти-FEP аргумент |
| [`city/`](./city/) | § 3.6 | $\sim 3 \cdot 10^{-30}$ (Сингапур, Детройт) | **Социо-техническая система**: структурно сложна, $\eta_L$ ничтожен; формула $I_v$ (2)+(2a) — на синтетике (без эмпирического рейтинга городов) |
| [`biosphere/`](./biosphere/) | § 3.7 | $\sim 10^{-18}$ (annual window, biomass-equiv. $I_{\text{pred}}$) | **Планетарная система**: единая Гея с гигантским ландауэровским бюджетом NPP |
| [`LLM-as-corp/`](./LLM-as-corp/) | § 3.4 | $\sim 10^{-25}$ (corp boundary) | **Расширение границы**: «модель + дата-центр + корпорация» даёт положительный, но катастрофически малый $\eta_L$ |

## Что не закрыто отдельным расчётом

Камертоны статьи, для которых $\eta_L = 0$ структурно (через числитель или знаменатель), не требуют worked example:

- **Солнце** (§ 3.1): $\eta_L = 0$ через числитель — нет каузально-релевантной среды.
- **Ураган и пламя** (§ 3.2): $\eta_L = 0$ через числитель — нет $I_{\text{pred}}$.
- **Кристалл** (§ 3.3): $\eta_L = 0$ через знаменатель — нет $E_{\text{actual}}$ при стационарной структуре.
- **LLM (узкая граница «веса в момент inference»)** (§ 3.4): $\eta_L = 0$ через числитель И знаменатель.

Полный воспроизводимый sweep по всем шести камертонам — будущая работа (см. paper § 3).

## Запуск всех расчётов

```bash
for dir in E-coli thermostat city biosphere LLM-as-corp; do
    echo "=== $dir ==="
    cd "$dir"
    python eta_L_*.py
    cd ..
done
```

В каждой папке также есть `expected_output.txt` для верификации воспроизводимости.

## Зависимости

Все расчёты используют **только стандартную библиотеку Python** (`math`, `statistics`). Никакого pip install не требуется.

Опциональные расширения (sensitivity dashboards, KSG-оценка взаимной информации, plotly-визуализации) — задел на будущую работу.

## Структура каждой папки

```
<system>/
├── README.md           — описание (англ., по умолчанию): модель, параметры, результаты, статус
├── README.ru.md        — то же на русском
├── eta_L_<system>.py   — основной расчёт
├── sensitivity.py      — sensitivity-анализ через Monte Carlo (там, где применимо)
└── expected_output.txt — captured output для верификации
```

## Статус

Расчёты — **методологическая калибровка**, демонстрирующая воспроизводимость камертонных оценок. Детальный экспериментальный дизайн, KSG-оценка $I_{\text{pred}}$ через измеримые временные ряды и полная sensitivity-карта sweep'а (§ 3.5, § 3.6) — будущая работа.
