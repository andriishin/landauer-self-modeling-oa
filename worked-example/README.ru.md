# Worked Examples for «Landauer Efficiency of Self-Modeling: An Operational Scale of Vitality»

Воспроизводимые расчёты $\eta_L$ и $I_v$ для пяти камертонных систем статьи. Каждая папка — самодостаточный пакет (Python-скрипт + README + эталонный вывод) с ссылкой на конкретную секцию статьи.

## Индекс

| Папка | Секция статьи | $\eta_L$ (центральная оценка) | Что демонстрирует |
|-------|----------------|-------------------------------|-------------------|
| [`E-coli/`](./E-coli/) | § 3.5 | $3 \cdot 10^{-8}$ (эксерг.), $3 \cdot 10^{-5}$ (вычисл.) | **Положительный случай**: бактерия проходит оба порога — центральный worked example статьи |
| [`thermostat/`](./thermostat/) | § 4.4 | $5 \cdot 10^{-18}$ тривиальная / $0$ управляющая | **FEP-граница**: формальная FEP-применимость без self-payment; центральный анти-FEP аргумент |
| [`city/`](./city/) | § 3.6 | $\sim 3 \cdot 10^{-30}$ (Сингапур, Детройт) | **Социо-техническая система**: структурно сложна, $\eta_L$ ничтожен; формула $I_v$ (2)+(2a) — на синтетике (без эмпирического рейтинга городов) |
| [`biosphere/`](./biosphere/) | § 3.7 | $\sim 10^{-22}$ (годовое окно, coding-bound $I_{\text{pred}}$) | **Планетарная система**: единая Гея с гигантским ландауэровским бюджетом NPP |
| [`LLM-as-corp/`](./LLM-as-corp/) | § 3.4 | $\sim 10^{-25}$ (граница корпорации) | **Расширение границы**: «модель + дата-центр + корпорация» даёт положительный, но катастрофически малый $\eta_L$ |

## Что не закрыто отдельным расчётом

Камертоны статьи, для которых $\eta_L = 0$ структурно (через числитель или знаменатель), не требуют worked example:

- **Солнце** (§ 3.1): $\eta_L = 0$ через числитель — нет каузально-релевантной среды.
- **Ураган и пламя** (§ 3.2): $\eta_L = 0$ через числитель — нет $I_{\text{pred}}$.
- **Кристалл** (§ 3.3): $\eta_L = 0$ через знаменатель — нет $E_{\text{actual}}$ при стационарной структуре.
- **LLM (узкая граница «веса в момент инференса»)** (§ 3.4): $\eta_L^{\text{int}}$ **не определён** через одновременный отказ $T_v^{\text{int}}$ и $C_v^{\text{int}}$.

Полный воспроизводимый скан по всем шести камертонам — будущая работа (см. статью § 3).

## Запуск всех расчётов

```bash
for dir in E-coli thermostat city biosphere LLM-as-corp; do
    echo "=== $dir ==="
    cd "$dir"
    python eta_L_*.py
    cd ..
done
```

В каждой папке также есть `expected_output.txt` для верификации воспроизводимости.

## Зависимости

Все расчёты используют **только стандартную библиотеку Python** (`math`, `statistics`). Никакого pip install не требуется.

Опциональные расширения (панели анализа чувствительности, KSG-оценка взаимной информации, plotly-визуализации) — задел на будущую работу.

## Структура каждой папки

```
<system>/
├── README.md           — описание (англ., по умолчанию): модель, параметры, результаты, статус
├── README.ru.md        — то же на русском
├── eta_L_<system>.py   — основной расчёт
├── sensitivity.py      — анализ чувствительности методом Монте-Карло (там, где применимо)
└── expected_output.txt — сохранённый вывод для верификации
```

## Статус

Расчёты — **методологическая калибровка**, демонстрирующая воспроизводимость камертонных оценок. Детальный экспериментальный дизайн, KSG-оценка $I_{\text{pred}}$ через измеримые временные ряды и полная карта скана чувствительности (§ 3.5, § 3.6) — будущая работа.
