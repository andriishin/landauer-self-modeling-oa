# Разобранный пример: η_v = I_pred / I_mem для хемотаксической петли *E. coli*

Центральный разобранный пример статьи: точный расчёт предсказательной доли

$$\eta_v = \frac{I_{\text{pred}}}{I_{\text{mem}}} = 1 - \nu \in [0, 1]$$

для бактериальной хемотаксической петли *E. coli*. Прилагается к § 3.5 +
Supplementary § S3.5 (Андрийшин, *Theory in Biosciences*, в подготовке). И
числитель, и знаменатель — **информационные** величины; энергетического
знаменателя нет. Бактерия — единственный камертон, где $\eta_v$ вычислена полностью
($S = 1$, $\eta_v \sim O(0{,}1)$).

## Содержимое

| Файл | Назначение |
|------|------------|
| `eta_ecoli.py` | Центральный расчёт: точные гауссовы $I_{\text{mem}}, I_{\text{pred}}, \eta_v, \nu$ через чистый stdlib-Cholesky log-определитель; печатает таблицу $(\tau_E,\beta_v)$, $\beta_v$-скан ностальгии и пример измеримости |
| `sensitivity.py` | Чувствительность по параметрам: как $\eta_v$ реагирует на дрейф $\tau_E$, окно прошлого $L$ и шум метилирования $r$ |
| `expected_output.txt` | Эталонный вывод `eta_ecoli.py` (LF) для верификации воспроизводимости |
| `expected_output_sensitivity.txt` | Эталонный вывод `sensitivity.py` (LF) |

Использует только стандартную библиотеку Python (`math`); без `numpy`, без
внешних зависимостей.

## Запуск

```bash
python eta_ecoli.py
python sensitivity.py
```

Вывод `eta_ecoli.py` воспроизводит `expected_output.txt` побитово (а
`sensitivity.py` — `expected_output_sensitivity.txt`). Числовые узлы
(в натах; согласованы с Supplementary § S3.5):

- $I_{\text{mem}}$ **конечна** (0,03–0,72 по прогону), $\eta_v \in [0,1]$ по data-processing inequality;
- характерное $\eta_v \sim O(0{,}1)$ (напр. $\tau_E=10, \beta_v=0{,}9$: $I_{\text{mem}}=0{,}50$, $I_{\text{pred}}=0{,}14$, $\eta_v=0{,}27$);
- $\beta_v$-скан ностальгии при $\tau_E = 5$: $\eta_v \approx 0{,}29 \to 0{,}035$ при $\beta_v = 0{,}3 \to 0{,}99$.

## Модель (Supplementary § S3.5)

Хемотаксис моделируется как **линейный адаптивный трекер** в пассивном
(иммобилизованном) FRET-протоколе Sourjik–Berg / Cluzel — клетка закреплена,
лиганд управляется внешне. Согласно main § 2.1 иммобилизация — **не упрощение, а
операционная реализация do-интервенции Пирла $\mathrm{do}(a)$**: экзогенный зажим
действия клетки разрывает обратную связь действие→среда и восстанавливает
марковость, которую предполагает граница $[0,1]$. Поэтому пассивная величина,
которую измеряет протокол, — это **интервенционная** предсказательная свежесть
$$\eta_v^{\mathrm{do}} = \frac{I\!\left(M_t;\, X_E^{(t,t+\tau]} \,\middle|\, \mathrm{do}(a)\right)}{I_{\text{mem}}} \in [0,1],$$
с числителем и знаменателем в *одном* режиме $\mathrm{do}(a)$ (свободно плывущая
клетка измеряла бы иную, активную величину, не ограниченную единицей). Три
элемента:

**(а) Сенсорный канал** — лог-концентрация лиганда $s_t$ как процесс AR(1) с
временем корреляции $\tau_E$ (дрейф среды), управляется микрофлюидикой.

**(б) Внутренний канал (память)** — уровень метилирования
$m_t = \sum_{j \ge 1} w_j\, s_{t-j} + \text{шум}$, leaky-интеграл прошлого лиганда
(линеаризованная адаптация Tu-типа) с весами $w_j = (1-\beta_v)\beta_v^{j-1}$, где
$\beta_v$ — память адаптации. Внутреннее состояние $M_t = m_t$, читается через FRET
(Sourjik–Berg 2002; Cluzel et al. 2000).

**(в) Окна** — окно прошлого $X_E^{\le t}$ длины $L$ и горизонт будущего
$X_E^{(t,t+\tau]}$.

Поскольку $(s_t, m_t)$ совместно гауссовы,

$$I_{\text{mem}} = I(M_t; X_E^{\le t}), \qquad I_{\text{pred}} = I(M_t; X_E^{(t,t+\tau]})$$

вычисляются **точно** через log-определители ковариационных матриц (здесь чистый
stdlib-Cholesky вместо `numpy.linalg.slogdet`). По data-processing inequality
$I_{\text{pred}} \le I_{\text{mem}}$, поэтому $\eta_v \in [0,1]$ по построению, а
$\nu = 1 - \eta_v$. Вычисляемая ниже $\eta_v$ — это, таким образом,
$\eta_v^{\mathrm{do}}$: пассивная, обусловленная $\mathrm{do}(a)$ свежесть, не
неограниченное активное значение.

## Измеримость

Ключевое: $I_{\text{mem}}$ зависит **только** от наблюдаемых ковариаций
$\text{Cov}(m, s_k)$ и $\text{Cov}(s_i, s_j)$ — обе оцениваемы из временны́х рядов
($m$ через FRET-метилирование, $s$ через микрофлюидику). Операциональный долг
информационного знаменателя поэтому **не хуже**, чем у калориметрической оценки
диссипации (прямой ответ на опасение, что информационную величину измерять
труднее, чем калориметрию).

## Ностальгия

Ностальгия $\nu = 1 - \eta_v$ растёт ($\eta_v$ падает), когда память адаптации $\beta_v$
**перерастает** дрейф среды $\tau_E$: клетка over-интегрирует устаревший лиганд, и
доля чистого балласта увеличивается. Это даёт ностальгии конкретное
фальсифицируемое прочтение — быстрее дрейф или медленнее адаптация → больше
балласта → ниже предсказательная доля. Число рецепторов/кластеров на $\eta_v$
напрямую **не** влияет: масштабирование числа независимых сенсорных каналов
поднимает числитель и знаменатель согласованно и сокращается в отношении.

## Эпистемический статус числа

$\eta_v \sim O(0{,}1)$ — **иллюстрация формы**, не количественная claim про реальную
*E. coli*: оно получено из минимальной линейной модели, цель которой — показать
измеримость и порядок. Точное экспериментальное отображение (какой FRET-readout
отождествить с $M_t$; окно для $X_E^{\le t}$; усечение кумулятива до конечной
$\tau_{\text{mem}}$) требует сверки с первоисточником и оставлено будущей работе.

## Статус

**Методологическая калибровка / proof-of-measurability**: не претендует на
калиброванное $\eta_v$ реальной *E. coli*, но показывает, что предсказательная доля
$\eta_v = I_{\text{pred}}/I_{\text{mem}}$ конечна, измерима из наблюдаемых корреляций
и $O(0{,}1)$ для хемотаксической памяти. Узкое место — экспериментальное
отображение FRET-readout ↔ $M_t$ и выбор окна прошлого, не число рецепторов.
Анализ чувствительности — в `sensitivity.py`.
