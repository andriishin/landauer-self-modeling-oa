========================================================================
Thermostat (bimetallic) -- structural verdict (main 4.4 / Supp S4.4)
========================================================================

Refounded quantity: eta_v = I_pred / I_mem in [0, 1]. Demarcation is
carried by the self-payment predicate S, not by an energetic-
efficiency ratio. For the thermostat S = 0, so eta_v is not defined.

Counterfactual ablation of the boundary (main 2.2):
  candidate                   inside?  why
  --------------------------------------------------------------
  bimetallic strip            yes      removing it breaks the loop
  relay contacts              yes      removing it breaks the loop
  heating element             no       removes heat source, not decision loop
  electric grid               no       external energy source
  external setpoint operator  no       does not disrupt regulation
  -> boundary = {bimetal, relay}

Surviving quantities (outlive the refounding; match main 4.4):
  internal dissipation   P_int  ~ 1e-06 W  (elastic relaxation + relay)
  predictive-info bound  I_pred <= log2(Delta T_room / Delta T_hyst):
    diurnal  Delta T_room ~ 10 K, hyst 1 K -> I_pred <= 3.32 bits (~3)
    seasonal Delta T_room ~ 30 K, hyst 1 K -> I_pred <= 4.91 bits (up to ~5)
  (external heater power P_ext ~ 1e+03 W -- outside)

Two self-consistent readings, both S = 0 via DIFFERENT failures:
  managing ('regulate the room'): condition (i) fails -- room heating
    is paid by the external grid, E_own about that task = 0 (T_v fails).
  trivial ('hold the bimetal's own shape'): condition (ii) fails --
    passive thermo-mechanical coupling forms no error-return loop; a
    shape deviation is not corrected by the system's own dissipation
    (S_v fails).

Verdict: S = 0 (not vital). eta_v = I_pred / I_mem is UNDEFINED -- a
numeric value would require an MI estimate of I_pred and I_mem of a
self-paying loop, which the thermostat does not possess.

Cross-check vs main 4.4 / Supplementary S4.4:
  S = 0 in both readings                          OK
  I_pred <= ~3 bits diurnal (up to ~5 seasonal)   OK (3.32 / 4.91)
  P_int ~ 1e-6 W                                  OK
