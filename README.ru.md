# Витальность как эффективность самооплаченного самомоделирования — открытые материалы

*([English version](./README.md))*

Открытые материалы к статье **«Витальность как эффективность самооплаченного самомоделирования»** (русский — основной язык; включён английский перевод).

Статья делает витальность измеримой как безразмерную *эффективность* — предсказательную долю самомодели системы

```
V(X) = S ∧ (η_v > 0),    η_v = I_pred / I_mem = 1 − ν ∈ [0, 1]
```

где `η_v` — доля удерживаемой системой памяти о среде, всё ещё предсказывающая будущее (её дополнение `ν` — *информационная ностальгия*), а `S` — предикат **самооплаты**: модель, диссипация, оплачивающая её удержание, и петля возврата ошибки принадлежат одной и той же физической системе. Двухэтапная процедура — структурный скрининг по композитному индексу `I_v`, затем верификация витальности парой `(S, η_v)` — применяется к шести камертонным случаям: Солнце, ураган, кристалл, бактерия (*E. coli*), большая языковая модель и крупный город (плюс биосфера как контрольный предел). Граница `η_v ≤ 1` следует безусловно из неравенства об обработке данных; единственный полностью вычисляемый случай — бактерия.

## Пересборка PDF

Готовые PDF включены для прямого чтения. Для пересборки из `.tex` нужен дистрибутив TeX с `pdflatex` и `bibtex` (TeX Live или MiKTeX). Файлы шаблона Springer Nature (`sn-jnl.cls`, `sn-*.bst`) вложены, ничего скачивать не требуется. Стиль ссылок — `sn-mathphys-ay` (автор–год). Стандартный цикл для любого документа, например `main`:

```bash
cd paper
pdflatex main.tex
bibtex   main
pdflatex main.tex
pdflatex main.tex
```

(замените `main` на `supplementary`, `main.ru` или `supplementary.ru` для остальных документов).

## Воспроизведение разобранных примеров

Каждый `worked-example/<система>/` самодостаточен и использует **только стандартную библиотеку Python** (никакого `pip install`):

```bash
cd worked-example/E-coli && python eta_ecoli.py
```

Вывод должен совпасть с зафиксированным `expected_output.txt` с точностью до округления чисел с плавающей точкой. Бактерия — единственный численно вычисляемый случай (`η_v = I_pred / I_mem` через точный гауссов логарифм определителя), а `sensitivity.py` прогоняет параметры модели (зафиксировано в `expected_output_sensitivity.txt`); остальные папки печатают **структурные вердикты** `(S, η_v)` — величина `η_v` там требует оценки взаимной информации и не утверждается. В каждой папке есть `README.md` (английский, по умолчанию) и `README.ru.md` (русский) с описанием модели, параметров с литературой, ожидаемых результатов и статуса. `worked-example/README.md` / `README.ru.md` — индекс.
